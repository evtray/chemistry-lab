## 1. Teoría Atómica del Núcleo

### 1.1 Partículas Subatómicas

| Partícula | Símbolo | Carga | Masa (uma) | Ubicación |
|-----------|---------|-------|------------|-----------|
| Protón    | p⁺      | +1    | 1.00728    | Núcleo    |
| Neutrón   | n⁰      | 0     | 1.00867    | Núcleo    |
| Electrón  | e⁻      | −1    | 0.000549   | Corteza   |

- **Número atómico (Z):** número de protones → define el elemento
- **Número másico (A):** protones + neutrones → A = Z + N
- **Neutrones (N):** N = A − Z

### 1.2 Modelos Atómicos (cronología)

| Año | Científico | Modelo |
|-----|-----------|--------|
| 1803 | Dalton | Esfera maciza indivisible |
| 1897 | Thomson | "Pudín de pasas" — electrones incrustados |
| 1911 | Rutherford | Núcleo positivo pequeño + electrones orbitando |
| 1913 | Bohr | Órbitas circulares cuantizadas |
| 1926 | Schrödinger | Nube electrónica / mecánica cuántica |

## 2. Isótopos, Número Atómico y Masa Atómica

### 2.1 Definición de Isótopos

Átomos del **mismo elemento** (mismo Z) con **diferente número de neutrones** (diferente A).

**Notación:** ^A_Z X

Ejemplo: ¹²₆C, ¹³₆C, ¹⁴₆C → todos son isótopos del Carbono

### 2.2 Masa Atómica Promedio (Peso Atómico)

m̄ = Σ mᵢ · fᵢ

donde mᵢ = masa del isótopo i y fᵢ = fracción de abundancia (decimal)

> Si la abundancia está en %, dividir entre 100 para obtener fracción.

### 2.3 Problemas Tipo

**Tipo A — Encontrar masa del isótopo faltante:** Se conocen 2 isótopos (masa y abundancia) y la masa promedio. La suma de abundancias = 1.

**Tipo B — Encontrar abundancia:** Se conocen todas las masas y la masa promedio.

**Tipo C — Identificar elemento:** Se conocen las masas (A) y neutrones totales de varios isótopos. Z = (A_total/n_isótopos) − (N_total/n_isótopos) promedio.

## 3. Energía Radiante y El Fotón

### 3.1 Propiedades de la Radiación Electromagnética

**c = λ · f**

| Símbolo | Significado | Unidad SI |
|---------|------------|-----------|
| c | Velocidad de la luz = 3×10⁸ m/s | m/s |
| λ | Longitud de onda | m |
| f (ν) | Frecuencia | Hz = s⁻¹ |

### 3.2 Espectro Electromagnético

Ondas de radio → Microondas → IR → Visible → UV → Rayos X → Rayos γ

**Visible:** 400 nm (violeta) → 700 nm (rojo)

### 3.3 Energía del Fotón — Teoría Cuántica de Planck

**E = h · f = (h · c) / λ**

| Constante | Valor |
|-----------|-------|
| h (Planck) | 6.626 × 10⁻³⁴ J·s |
| c | 3 × 10⁸ m/s |
| 1 eV | 1.602 × 10⁻¹⁹ J |

**Energía total de N fotones:** E_total = N · h · f

**Potencia:** P = E_total / t = (N · h · f) / t

## 4. Átomo de Hidrógeno y Transiciones Electrónicas

### 4.1 Modelo de Bohr — Hidrógeno (Z=1)

**E_n = −13.6 / n² eV** (n = 1, 2, 3, ...)

**r_n = n² · a₀** con a₀ = 5.292 × 10⁻¹¹ m (radio de Bohr)

**Regla de oro:** E_k = −E_n y V = 2E_n

### 4.2 Átomo Hidrogenoide (Z > 1)

**E_n = −13.6 · Z² / n² eV**

**r_n = n² · a₀ / Z**

### 4.3 Transiciones Electrónicas

- n_f > n_i → **Absorbe** fotón
- n_f < n_i → **Emite** fotón

**E_fotón = |E_nf − E_ni|**

**Series espectrales:** Lyman (nf=1, UV), Balmer (nf=2, visible), Paschen (nf=3, IR), Brackett (nf=4, IR lejano)

### 4.4 Ionización

**E_ionización = 13.6 · Z² / n_i² eV** (desde n_i hasta ∞)

## 5. Efecto Fotoeléctrico

### 5.1 Condición

**E_fotón ≥ φ** (función de trabajo del metal)

### 5.2 Ecuación de Einstein

**h · f = φ + E_k^max**

**Frecuencia umbral:** f₀ = φ / h

**Longitud de onda umbral:** λ₀ = hc / φ

### 5.3 Potencial de Frenado

**E_k^max = e · V_f** → **V_f = E_k / e**

### 5.4 Funciones de Trabajo (eV)

## 6. Números Cuánticos

| Número | Símbolo | Valores | Describe |
|--------|---------|---------|----------|
| Principal | n | 1, 2, 3, ... | Nivel / tamaño orbital |
| Azimutal | l | 0, 1, ..., n−1 | Forma del orbital |
| Magnético | m_l | −l, ..., +l | Orientación |
| Espín | m_s | +½ ó −½ | Giro del electrón |

**Subniveles:** s (l=0), p (l=1), d (l=2), f (l=3)

**Electrones máx. por nivel n:** 2n²

## 7. Configuración Electrónica

### 7.1 Orden de llenado (Aufbau)

1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → ...

### 7.2 Regla de Hund

En orbitales degenerados: **uno por orbital** con mismo espín antes de emparejar.

### 7.3 Principio de Exclusión de Pauli

Máximo **2 electrones** por orbital (espines opuestos).

### 7.4 Excepciones

Cr: [Ar] 4s¹ 3d⁵ | Cu: [Ar] 4s¹ 3d¹⁰ | Mo: [Kr] 5s¹ 4d⁵

### 7.5 Paramagnético vs Diamagnético

- **Paramagnético:** electrones desapareados → atraído por imán
- **Diamagnético:** todos apareados → repelido levemente

### 7.6 Carga Nuclear Efectiva

**Z* = Z − σ** (σ = constante de apantallamiento, reglas de Slater)
